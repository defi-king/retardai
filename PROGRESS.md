## Setup Requirements

1. Environment Variables needed:
   ```
   TAVILY_API_KEY=your_tavily_api_key
   OPENROUTER_API_KEY=your_openrouter_api_key
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
